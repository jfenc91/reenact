Why reenact?

Screen recorders capture pixels. reenact captures intent — and reenacts it perfectly every time.

Bezier mouse curves

Mouse movements follow cubic Bezier curves with randomized control points. Every recording looks natural, never robotic.

Natural typing

Characters appear at a realistic WPM with variable delay per keystroke, pauses between words, and occasional micro-hesitations.

Smooth scrolling

Scroll events are broken into incremental steps with jitter — mimicking how a human uses a trackpad or mouse wheel.

Real browser video

Powered by Playwright's Chromium. The output is a real WebM video of an actual browser rendering your page — not a simulation.

Single command

One command to install: npm i -g reenact. Chromium downloads automatically. Zero config.

Reproducible

YAML scripts are version-controlled, diffable, and repeatable. Re-record after a redesign in seconds, not hours.

How it works

1

Write a YAML script

Describe the navigation flow in plain YAML — what to click, type, scroll, and wait for. No code, no test framework.

2

reenact drives a real browser

Playwright launches headless Chromium. Each step is executed with human-like timing and motion while recording video.

3

Get a WebM video

The output is a standard video file you can embed in docs, READMEs, presentations, or marketing pages.

Getting started

Requirements

Node.js 18+

Install

$ npm install -g reenact

Your first recording

Create demo.yaml:

url: https://example.com
steps:
  - wait: 1s
  - click: "h1"
  - scroll: down

Run it:

$ reenact demo.yaml -o demo.webm

Examples

Copy any of these, swap the URLs and selectors for your app, and run.

Login flow

Form interaction
View YAML script 
# Record a realistic login interaction
name: Login Flow
url: https://the-internet.herokuapp.com/login
viewport:
  width: 1280
  height: 720

steps:
  - wait: 1s

  # Type credentials naturally
  - type:
      selector: "#username"
      text: "tomsmith"
      wpm: 65

  - wait: 300ms

  - type:
      selector: "#password"
      text: "SuperSecretPassword!"
      wpm: 55

  - wait: 500ms
  - click: "button[type='submit']"
  - wait: 2s
  - scroll: down

Action reference

Every step you can use in a script.

click

- click: "#btn"

Smooth mouse move to element, then click at a random point within its bounds.

type

- type: selector: "#input" text: "hello" wpm: 70

Type character by character at natural speed. wpm controls pace (default 70).

hover

- hover: ".menu-item"

Move the cursor over an element without clicking. Great for tooltips and menus.

scroll

- scroll: down - scroll: direction: up amount: 500

Smooth incremental scroll. Direction is up or down, amount in pixels (default 300).

wait

- wait: 2s - wait: "#loaded"

Pause for a duration (ms, s, m) or wait until a selector appears.

wait_for

- wait_for: selector: ".modal" state: visible timeout: 10s

Wait for an element to reach a state: visible, attached, hidden, detached.

navigate / goto

- navigate: https://example.com

Navigate to a new URL. Waits for DOM content to load.

key / press

- key: Enter - key: Control+A

Press a keyboard key or combination.

fill

- fill: selector: "#bio" text: "long content..."

Instantly fill a field (non-humanized). Use for long text where typing would be slow.

select

- select: selector: "#country" value: "US"

Choose a value from a <select> dropdown.

clear

- clear: "#search"

Clear the contents of an input field.

screenshot

- screenshot: output.png

Capture a screenshot mid-recording. Useful for debugging or docs.

back / forward / reload

- back - forward - reload

Browser history navigation and page reload.

move_to

- move_to: x: 500 y: 300 duration: 600

Move cursor to exact coordinates with smooth bezier motion.
